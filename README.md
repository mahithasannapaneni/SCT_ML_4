# Hand Gesture Recognition using K-Nearest Neighbors (KNN)

## Project Overview
This project implements a Hand Gesture Recognition system using the K-Nearest Neighbors (KNN) machine learning algorithm. The model classifies hand gesture images into different categories such as Rock, Paper, and Scissors.

The project demonstrates how machine learning and image processing techniques can be used for gesture-based recognition systems and human-computer interaction applications.

---

## Technologies Used
- Python
- NumPy
- Matplotlib
- OpenCV
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Algorithm
- K-Nearest Neighbors (KNN)

---

## Dataset Used
Rock-Paper-Scissors Hand Gesture Dataset

Gesture Categories:
- Rock
- Paper
- Scissors

The dataset contains image samples of different hand gestures used for training and testing the model.

---

## Project Workflow
1. Import required libraries
2. Load hand gesture image dataset
3. Resize and preprocess images
4. Convert images into feature vectors
5. Split dataset into training and testing sets
6. Train KNN classification model
7. Predict gesture classes
8. Evaluate model accuracy
9. Visualize prediction results

Run all notebook cells sequentially.
